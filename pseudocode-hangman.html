Psuedocode for hangman:

  1. User clicks any key to begin (related text hides).
  2. Computer chooses current word from array of hidden words
    1. Current word area populates with number of underscores matching number of letters in word
  3. User guesses letter
    1. Guessed letter appears in Already Guessed area, comma-separated
    2. If user selects letter already guessed, nothing happens. (Maybe "bad" sound.)
    3. If user guesses correctly, letter populates in current word area in proper sequence
    4. If user guesses incorrectly, number of guesses decreases by one.
  4. If user correctly guesses word, happy event happens:
    1. Correct word displays somewhere
    2. Song?
    3. Game elements reset for next hidden word.
  5. If number of guesses hits zero, then it's the end of game, "press any key" text reappears, and any key press allows game to begin anew.
